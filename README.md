## Objective

The objective of this task is to clean and preprocess a raw dataset by identifying and handling common data quality issues such as missing values, duplicate records, inconsistent data formats, and incorrect data types. The cleaned dataset is prepared for further analysis.

## Data Cleaning Steps

- Checked for missing values.
- Handled missing values using appropriate methods (fill or remove).
- Removed duplicate records.
- Standardized text values (e.g., Gender, Country).
- Converted date columns into a consistent format.
- Renamed column headers for better readability.
- Corrected data types where required.
- Verified the cleaned dataset for consistency and accuracy.
